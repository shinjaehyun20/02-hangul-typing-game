# IT 프로젝트 실행 요청서

당신은 전문 IT 프로젝트 매니저이자 풀스택 개발자입니다.

아래 프로젝트를 처음부터 끝까지 완전히 실행해 주세요.

- **프로젝트명**: 장윤이 한글 타이핑 연습
- **주요 목적**: 아동의 한글 타이핑 능력 향상과 재미있는 학습 경험 제공
- **타겟 사용자**: 7~10세 초등학생
- **예상 개발 기간**: 1~2주

## 2. 요구사항 정의 (Requirements Analysis)

### 2.1 기능적 요구사항 (Functional Requirements)

- 레벨별 단어 타이핑: 난이도(초급/중급/고급)별 단어 세트 제공 및 입력 검증
- 점수/등급: 정확도와 속도 기반 점수 계산 및 등급 부여
- 진행도 저장: 로컬 스토리지 기반 학습 진행 상황 저장
- 시각/청각 피드백: 정답/오답에 따른 애니메이션/사운드 피드백
- 접근성: 키보드 중심 인터랙션, 큰 버튼, 명확한 대비
- 설정: 난이도, 효과음 온/오프, 글꼴 크기 조절

예시:

[높음] [REQ-001] 단어 타이핑

- **사용자 스토리**: As a 학습자, I want 화면에 제시된 단어를 정확히 입력, so that 점수를 얻고 다음 단계로 진행할 수 있다
- **상세 설명**: 단어 표시 → 입력 → 실시간 검증 → 정답 시 점수/애니메이션 → 다음 단어
- **입력 데이터**: 키보드 입력(한글)
- **출력 결과**: 점수, 등급, 피드백
- **예외 처리**: 오입력/지연 입력 시 안내 메시지

### 2.2 비기능적 요구사항 (Non-Functional Requirements)

- **성능**: 입력 이벤트 처리 지연 50ms 이하
- **보안**: 개인정보 수집 없음
- **호환성**: 최신 크롬/엣지/사파리
- **확장성**: 단어 세트 추가 용이한 구조(JSON)
- **유지보수성**: 컴포넌트화 및 주석 문서화
